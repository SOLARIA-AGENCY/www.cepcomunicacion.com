Requisitos Técnicos

Equipo Requerido

  • • Ordenador o tablet con conexión a internet
  • • Navegador web actualizado (Chrome, Firefox, Safari)
  • • Altavoces o auriculares
  • • Webcam (opcional para tutorías)

Conexión

  • • Conexión a internet estable
  • • Mínimo 5 Mbps de velocidad
  • • Conexión por cable o WiFi
  • • Datos móviles no recomendados
